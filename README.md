# SAS Programming Essentials 1

This repository contains practical exercises and projects developed while completing the SAS Programming Essentials 1 course using SAS Viya for Learners.

## Objectives

- Develop foundational SAS programming skills
- Understand DATA step processing
- Apply PROC procedures for analysis
- Perform data import/export operations
- Generate basic analytical reports

## Repository Structure

### /data
Contains datasets used for exercises and projects.

### /code
Contains SAS programs (.sas files) organized by lesson and topic.

### /output
Contains exported reports, generated tables, and analytical outputs.

## Skills Demonstrated

- DATA Step processing
- LIBNAME statement usage
- SET and MERGE operations
- PROC PRINT, PROC MEANS, PROC FREQ
- Basic data cleaning and transformation
- Output delivery system (ODS)

## Environment

Developed using:
- SAS Viya for Learners
- SAS Studio (web interface)
